## Actions

* hierarchy
  * ADD_CHILD parent, child
  * DELETE_NODE node
  * MOVE_NODE node, new_parent
* mappings
  * ADD_MAPPING input, output
  * REMOVE_MAPPING mapping

## Path

1. check hierarchy
2. render things
